# Hybrid Quality Monitoring System with SPC and Machine Learning _混合式品質監控系統（統計製程管制 + 機器學習）

---

## Overview
This project demonstrates a hybrid approach for quality monitoring in manufacturing processes by combining **Statistical Process Control (SPC)** and **Machine Learning (Random Forest)**.  
The system can automatically detect anomalies in sensor measurements, evaluate SPC rules (3σ method), and predict failure outcomes.

---


## Features
- 📂 Load and clean CSV data  
- 📊 SPC 3σ anomaly detection  
- 🤖 Random Forest ML model for Pass/Fail prediction  
- 📈 Sensor visualization for exploratory data analysis (EDA)  

---

## Dataset
The dataset used in this project is from **Kaggle: SECOM Manufacturing Data**
